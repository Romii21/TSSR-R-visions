### 2.1 Les rôles Active Directory

|Rôle|Acronyme|Description|
|---|---|---|
|Active Directory Domain Services|**AD DS**|Rôle principal - Gestion du domaine et de l'annuaire|
|Active Directory Certificate Services|**AD CS**|Gestion des certificats et clés de chiffrement|
|Active Directory Federation Services|**AD FS**|Portail SSO (Single Sign-On)|
|Active Directory Rights Management Services|**AD RMS**|Gestion des autorisations sur les fichiers|
|Active Directory Lightweight Directory Services|**AD LDS**|Service d'annuaire sans domaine|

**AD DS** est le rôle fondamental qui permet :

- Création et gestion d'un domaine
- Authentification et autorisation des utilisateurs
- Application des stratégies de groupe (GPO)
- Contrôle d'accès aux ressources

### 2.2 Protocole LDAP

**LDAP** est le protocole standardisé utilisé par AD pour :

- Accéder aux données de l'annuaire
- Effectuer des recherches structurées
- Modifier les objets de manière contrôlée

**Avantages :**

- Standard ouvert et compatible avec de nombreux systèmes
- Interopérabilité avec Linux, OpenLDAP, solutions SSO
- Méthode standard d'interrogation de l'annuaire

**Fichiers LDIF (LDAP Data Interchange Format)** :

- Permettent d'importer/exporter des données
- Modifient directement la structure de l'AD
- ⚠️ Méthode risquée à utiliser avec précaution

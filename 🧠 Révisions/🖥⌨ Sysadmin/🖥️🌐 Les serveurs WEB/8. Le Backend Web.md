### Définition

Le **backend** est un programme qui génère la réponse HTTP **en temps réel** au lieu de simplement servir un fichier statique.

### Technologies backend populaires

|Langage/Framework|Description|Exemple d'utilisation|
|---|---|---|
|**PHP**|Langage scriptable, intégré aux serveurs web|WordPress, Laravel|
|**Java**|Spring Boot, Java EE, serveurs d'application|Applications d'entreprise|
|**Node.js**|JavaScript côté serveur|APIs REST, applications temps réel|
|**Python**|Django, Flask|Applications web, APIs|
|**Ruby**|Ruby on Rails|Applications web rapides|
|**ASP.NET**|Framework Microsoft|Applications Windows|

### Architecture moderne

```
Client ──> Serveur Web ──> Backend ──> Base de données
                           │
                           └──> Génère le HTML/JSON
                                dynamiquement
```

Le web évolue vers des **applications web** complètes plutôt que de simples sites statiques.

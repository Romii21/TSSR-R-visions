### Définition

**HTTPS** = HTTP over TLS (Transport Layer Security)

|Caractéristique|Description|
|---|---|
|**Port standard**|443|
|**Protocole**|HTTP + TLS (chiffrement)|
|**Confidentialité**|Chiffrement des données transmises|
|**Authentification**|Certificat X.509 du serveur|
|**Autorité de certification**|Nécessaire (ex : Let's Encrypt)|

### Pourquoi HTTPS est indispensable ?

- Protection des données sensibles (mots de passe, cartes bancaires)
- Authentification du serveur
- Intégrité des données
- Meilleur classement SEO sur Google
- Confiance des utilisateurs

### Technologies complémentaires

|Technologie|Rôle|
|---|---|
|**HSTS**|HTTP Strict Transport Security - Force l'utilisation de HTTPS|
|**HPKP**|HTTP Public Key Pinning - Validation des certificats|
|**CSP**|Content Security Policy - Contrôle des ressources chargées|

### Structure d'une URL

```
schéma://[user:pass@]autorité[:port]/chemin?requête#fragment
```

### Décomposition des composants

|Composant|Description|Exemple|Obligatoire|
|---|---|---|---|
|**Schéma**|Protocole utilisé|`https`, `http`, `ftp`, `mailto`|✅ Oui|
|**Autorité**|Domaine ou adresse IP|`www.example.com`|✅ Oui|
|**User:pass**|Authentification|`user:password@`|❌ Non|
|**Port**|Port de connexion|`:443`, `:80`, `:8080`|❌ Non (par défaut selon schéma)|
|**Chemin**|Emplacement de la ressource|`/dossier/page.html`|❌ Non (peut être vide)|
|**Requête**|Paramètres clé=valeur|`?id=123&lang=fr`|❌ Non|
|**Fragment**|Ancre dans la page|`#section2`|❌ Non|

### Exemples d'URL

```
https://www.example.com/blog/article.html?id=5&category=tech#introduction
│     │ │              ││                 │                    │
│     │ │              ││                 │                    └─ Fragment (ancre)
│     │ │              ││                 └─ Paramètres de requête
│     │ │              │└─ Chemin vers la ressource
│     │ │              └─ Nom de domaine
│     │ └─ Sous-domaine
│     └─ Schéma (protocole)
└─ URL complète
```

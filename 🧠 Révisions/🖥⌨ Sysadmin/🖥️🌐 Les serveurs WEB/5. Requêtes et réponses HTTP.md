### Structure d'une requête HTTP

```http
GET /index.html HTTP/1.1
Host: www.example.com
User-Agent: Mozilla/5.0
Cookie: session=abc123
```

|Élément|Description|
|---|---|
|**Ligne de requête**|Méthode + URL + Version HTTP|
|**Méthode**|GET, POST, PUT, DELETE, etc.|
|**Entêtes**|Host (obligatoire), Cookie, User-Agent, etc.|
|**Corps**|Données envoyées (pour POST, PUT)|

### Structure d'une réponse HTTP

```http
HTTP/1.1 200 OK
Content-Type: text/html
Content-Length: 1234
Server: nginx/1.18.0

<!DOCTYPE html>
<html>...
```

|Élément|Description|
|---|---|
|**Ligne de statut**|Version HTTP + Code + Description|
|**Code de statut**|200 (OK), 404 (Not Found), 500 (Server Error), etc.|
|**Entêtes**|Content-Type, Content-Length, Server, etc.|
|**Corps**|Ressource demandée (HTML, JSON, image, etc.)|

### Principaux codes de statut HTTP

|Code|Signification|Description|
|---|---|---|
|**2xx**|Succès|La requête a réussi|
|**200**|OK|Succès standard|
|**3xx**|Redirection|Le client doit effectuer une action supplémentaire|
|**301**|Moved Permanently|Redirection permanente|
|**4xx**|Erreur client|Problème dans la requête|
|**404**|Not Found|Ressource introuvable|
|**5xx**|Erreur serveur|Le serveur a rencontré une erreur|
|**500**|Internal Server Error|Erreur interne du serveur|

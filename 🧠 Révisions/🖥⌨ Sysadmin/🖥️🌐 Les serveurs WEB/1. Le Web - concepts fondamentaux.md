### Définition

Le Web (World Wide Web) est un **système de publication hypertexte** fonctionnant sur les réseaux IP, créé au début des années 1990 par **Tim Berners-Lee** et **Robert Cailliau**.

### Les 3 composants essentiels

|Composant|Signification|Description|
|---|---|---|
|**HTTP**|HyperText Transfer Protocol|Protocole de transfert de données|
|**URL**|Uniform Resource Locator|Système d'adressage des ressources|
|**HTML**|HyperText Markup Language|Langage de balisage pour créer des pages web|

Le Web est devenu une **plateforme majeure de développement d'applications** qui s'appuie sur ces trois technologies fondamentales.

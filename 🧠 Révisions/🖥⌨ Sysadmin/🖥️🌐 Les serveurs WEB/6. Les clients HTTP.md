### Navigateurs web

Les **navigateurs web** sont les clients HTTP les plus courants. Ils combinent plusieurs fonctions :

|Fonction|Description|
|---|---|
|**Client HTTP**|Émet les requêtes HTTP|
|**Interprète HTML/CSS**|Affiche les pages web|
|**Environnement JavaScript**|Exécute le code JavaScript|
|**API Web**|DOM, Fetch, Canvas, WebGL, etc.|

### Exemples de clients HTTP

|Type|Exemples|Usage|
|---|---|---|
|**Navigateurs**|Chrome, Firefox, Safari, Edge|Navigation web classique|
|**Outils CLI**|`curl`, `wget`|Téléchargement, tests, scripts|
|**Bibliothèques**|Axios, Requests (Python)|Développement d'applications|
|**Outils de test**|Postman, Insomnia|Tests d'API REST|

#### Commandes CLI courantes

```bash
# Télécharger une page web avec curl
curl https://www.example.com

# Afficher les entêtes HTTP
curl -I https://www.example.com

# Télécharger un fichier avec wget
wget https://www.example.com/fichier.pdf

# Effectuer une requête POST
curl -X POST -d "param=value" https://api.example.com/endpoint
```

Le NAT repose sur **3 critères** qui peuvent être combinés :

### 1️⃣ Sens de traduction

- **NAT source** : l'adresse source est modifiée (sortie vers Internet)
- **NAT destination** : l'adresse destination est modifiée (publication de services)

### 2️⃣ Mode d'association

- **Statique** : association fixe, connue à l'avance (serveurs)
- **Dynamique** : association temporaire, créée à la demande (clients)

### 3️⃣ Niveau de traduction

- **Adresse IP uniquement** : 1 IP interne ↔ 1 IP externe (NAT 1:1)
- **Adresse IP + port** : plusieurs flux via une même IP publique (PAT/NAPT)

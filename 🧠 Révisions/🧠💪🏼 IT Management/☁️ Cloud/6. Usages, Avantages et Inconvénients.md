### 6.4 Sécurisation de l'accès aux services

#### Authentification Multi-Facteurs (MFA)

**Définition** : Combinaison de plusieurs facteurs d'authentification

**Méthode** :

1. Quelque chose que vous connaissez (mot de passe)
2. Quelque chose que vous possédez (code SMS, app, token physique)

**Importance** : Obligatoire pour tous les comptes administrateurs

#### RBAC (Role-Based Access Control)

**Principe du moindre privilège** : Donner uniquement les droits nécessaires

**Rôles standards** :

- **Lecteur** : Lecture seule
- **Contributeur** : Lecture et modification (sauf gestion des accès)
- **Propriétaire** : Tous les droits

#### Accès Conditionnel

**Exemple de politique** :

- SI géolocalisation = pays étranger
- ET utilisateur = Admin
- ALORS blocage des accès

**Autres conditions** :

- Bloquer certains pays
- Exiger MFA hors réseau d'entreprise
- Bloquer les appareils non conformes

## 💻 PARTIE 4 : Commandes Côté CLIENT

### Configuration et Tests

| Commande                                              | Utilité                                             |
| ----------------------------------------------------- | --------------------------------------------------- |
| `ipconfig /all`                                       | Voir toute la configuration réseau du client        |
| `ipconfig /release`                                   | Libérer l'adresse IP actuelle                       |
| `ipconfig /renew`                                     | Demander une nouvelle adresse IP au DHCP            |
| `ipconfig /flushdns`                                  | Vider le cache DNS local                            |
| `ipconfig /displaydns`                                | Afficher le contenu du cache DNS                    |
| `ping serveur.wilder.lan`                             | Tester la résolution DNS + connectivité             |
| `nslookup serveur.wilder.lan`                         | Tester uniquement la résolution DNS                 |
| `nslookup serveur.wilder.lan 172.16.10.5`             | Tester la résolution avec un serveur DNS spécifique |
| `Test-NetConnection -ComputerName serveur.wilder.lan` | Test avancé de connectivité (PowerShell)            |

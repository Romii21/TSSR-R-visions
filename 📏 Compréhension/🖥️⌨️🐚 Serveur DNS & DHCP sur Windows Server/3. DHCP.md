## 📡 PARTIE 3 : Installation et Configuration DHCP

### Étape 1 : Installation du rôle DHCP

|Commande|Utilité|
|---|---|
|`Install-WindowsFeature -Name DHCP -IncludeManagementTools`|Installer le rôle DHCP avec les outils|
|`Get-WindowsFeature DHCP`|Vérifier que DHCP est installé|
|`Add-DhcpServerInDC -DnsName "serveur.wilder.lan" -IPAddress 172.16.10.5`|Autoriser le serveur DHCP dans Active Directory (si domaine)|
|`Get-Service DHCPServer`|Vérifier l'état du service DHCP|
|`Restart-Service DHCPServer`|Redémarrer le service DHCP|

### Étape 2 : Création d'une étendue DHCP (Scope)

**Via l'interface graphique :**

1. Ouvrir **DHCP Manager** (Tools → DHCP)
2. Développer le serveur → IPv4 → Clic droit → New Scope
3. Name: `Atelier_LAN_172.20.0.0/24`
4. Start IP: `172.16.10.50` / End IP: `172.16.10.100`
5. Subnet mask: `255.255.255.0` (ou Length: 24)
6. Configurer les options (gateway, DNS) → Finish

**Via PowerShell :**

|Commande|Utilité|
|---|---|
|`Add-DhcpServerv4Scope -Name "Atelier_LAN" -StartRange 172.16.10.50 -EndRange 172.16.10.100 -SubnetMask 255.255.255.0 -State Active`|Créer une étendue DHCP (plage d'IP)|
|`Get-DhcpServerv4Scope`|Lister toutes les étendues DHCP|
|`Set-DhcpServerv4Scope -ScopeId 172.16.10.0 -State Active`|Activer une étendue|

### Étape 3 : Configuration des options DHCP

|Commande|Utilité|Option DHCP|
|---|---|---|
|`Set-DhcpServerv4OptionValue -ScopeId 172.16.10.0 -Router 172.16.10.1`|Configurer la passerelle par défaut|**003 - Router**|
|`Set-DhcpServerv4OptionValue -ScopeId 172.16.10.0 -DnsServer 172.16.10.5`|Configurer le serveur DNS|**006 - DNS Servers**|
|`Set-DhcpServerv4OptionValue -ScopeId 172.16.10.0 -DnsDomain "wilder.lan"`|Configurer le nom de domaine DNS|**015 - DNS Domain Name**|
|`Get-DhcpServerv4OptionValue -ScopeId 172.16.10.0`|Voir toutes les options configurées|Vérification|

### Étape 4 : Réservations DHCP (IP fixe pour un client spécifique)

|Commande|Utilité|
|---|---|
|`Add-DhcpServerv4Reservation -ScopeId 172.16.10.0 -IPAddress 172.16.10.10 -ClientId "08-00-27-0E-78-87" -Name "Windows_10"`|Réserver une IP pour une adresse MAC spécifique|
|`Get-DhcpServerv4Reservation -ScopeId 172.16.10.0`|Lister toutes les réservations|
|`Remove-DhcpServerv4Reservation -ScopeId 172.16.10.0 -IPAddress 172.16.10.10`|Supprimer une réservation|

### Étape 5 : Gestion des baux DHCP

|Commande|Utilité|
|---|---|
|`Get-DhcpServerv4Lease -ScopeId 172.16.10.0`|Voir tous les baux DHCP actifs (clients connectés)|
|`Remove-DhcpServerv4Lease -ScopeId 172.16.10.0 -IPAddress 172.16.10.20`|Libérer un bail DHCP manuellement|

## 📦 PARTIE 2 : Installation des Services

### Installation de Bind9 (DNS) et ISC-DHCP-Server

|Commande|Utilité|
|---|---|
|`sudo apt update`|Mettre à jour la liste des paquets disponibles|
|`sudo apt upgrade -y`|Mettre à jour tous les paquets installés|
|`sudo apt install bind9 bind9-utils bind9-doc -y`|Installer Bind9 et ses outils|
|`sudo apt install isc-dhcp-server -y`|Installer le serveur DHCP|
|`dpkg -l \| grep bind9`|Vérifier que Bind9 est installé|
|`dpkg -l \| grep isc-dhcp-server`|Vérifier que ISC-DHCP est installé|

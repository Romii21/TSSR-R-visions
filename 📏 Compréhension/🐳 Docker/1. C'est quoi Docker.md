
Docker est un outil qui permet d'**emballer une application avec tout ce dont elle a besoin** (code, bibliothèques, configurations) dans un **conteneur**.

### Analogie simple

Imagine que tu veux envoyer un objet fragile :

- **Sans Docker** : tu envoies l'objet seul, il risque de casser en route
- **Avec Docker** : tu mets l'objet dans une boîte avec tout le matériel de protection, il arrive intact partout

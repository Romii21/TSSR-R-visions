### Exemple 1 : Serveur web simple

```bash
# Télécharger et lancer nginx sur le port 8080
docker run -d -p 8080:80 nginx

# Vérifier que ça tourne
docker ps

# Ouvrir http://localhost:8080 dans ton navigateur
```

### Exemple 2 : Base de données MySQL

```bash
# Lancer MySQL avec un mot de passe root
docker run -d \
  -e MYSQL_ROOT_PASSWORD=monmotdepasse \
  -p 3306:3306 \
  mysql

# Se connecter à MySQL
docker exec -it <id-conteneur> mysql -u root -p
```

### Exemple 3 : Tester un environnement

```bash
# Lancer Ubuntu et explorer
docker run -it ubuntu bash

# Une fois dans le conteneur
root@xyz:/# ls
root@xyz:/# apt update
root@xyz:/# exit  # pour sortir
```

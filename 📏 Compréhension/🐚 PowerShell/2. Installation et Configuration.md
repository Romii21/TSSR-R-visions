### Outils disponibles

|Outil|Description|
|---|---|
|**PowerShell ISE**|Éditeur natif Windows avec console intégrée|
|**Terminal PowerShell**|Console en ligne de commande|
|**Visual Studio Code**|Éditeur externe recommandé avec extensions|

### Lancer PowerShell ISE

1. Menu Démarrer → Rechercher "Windows PowerShell ISE"
2. Interface :
    - **Partie bleue (bas)** : Terminal/Console
    - **Partie blanche (haut)** : Éditeur de script
3. Menu **Afficher** → "Afficher le volet de script" pour basculer

### Premier script

powershell

```powershell
Write-Host "Hello World !"
```

**Bonnes pratiques** :

- Extension : `.ps1`
- Commentaires avec `#`
- Noms de variables explicites
- Convention : **PascalCase**

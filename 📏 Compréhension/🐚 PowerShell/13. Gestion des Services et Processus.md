### Gestion des services

**Lister les services** :

powershell

```powershell
Get-Service                              # Tous les services
Get-Service | Where-Object {$_.Status -eq "Running"}  # Seulement actifs
Get-Service -Name "WalletService"        # Service spécifique
```

**Démarrer/Arrêter** :

powershell

```powershell
Start-Service -Name "WalletService"
Stop-Service -Name "WalletService"
Restart-Service -Name "WalletService"
```

**Modifier le mode de démarrage** :

powershell

```powershell
Set-Service -Name "WalletService" -StartupType Automatic  # Automatique
Set-Service -Name "WalletService" -StartupType Manual     # Manuel
Set-Service -Name "WalletService" -StartupType Disabled   # Désactivé
```

**Modifier l'état** :

powershell

```powershell
Set-Service -Name "WalletService" -Status Running
Set-Service -Name "WalletService" -Status Stopped
```

### Gestion des processus

**Lister les processus** :

powershell

```powershell
Get-Process                                    # Tous les processus
Get-Process -Name "CalculatorApp"              # Processus spécifique
Get-Process | Where-Object {$_.Name -like "*calc*"}
```

**Démarrer un processus** :

powershell

```powershell
Start-Process -FilePath "calc.exe"
Start-Process -FilePath "notepad.exe" -ArgumentList "C:\fichier.txt"
```

**Arrêter un processus** :

powershell

```powershell
Stop-Process -Name "CalculatorApp"
Stop-Process -Id 1234                  # Par ID
Stop-Process -Name "notepad" -Force    # Forcer
```

**Informations détaillées** :

powershell

```powershell
Get-Process -Name "CalculatorApp" -FileVersionInfo | Format-List
```

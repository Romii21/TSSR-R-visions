### Concepts

**Job** : Tâche exécutée en arrière-plan de manière asynchrone

**Avantages** :

- Exécuter des tâches longues sans bloquer le script principal
- Traiter plusieurs tâches simultanément
- Améliorer les performances

### Commandes de gestion

| Commande      | Description             |
| ------------- | ----------------------- |
| `Start-Job`   | Démarre un job          |
| `Get-Job`     | Liste les jobs          |
| `Receive-Job` | Récupère le résultat    |
| `Wait-Job`    | Attend la fin d'un job  |
| `Stop-Job`    | Arrête un job           |
| `Remove-Job`  | Supprime un job         |
| `Suspend-Job` | Suspend un job          |
| `Resume-Job`  | Reprend un job suspendu |
### Exemple d'exécution parallèle

**Exécution séquentielle** (30 secondes) :

powershell

```powershell
Write-Host "Début : $(Get-Date)"
Start-Sleep -Seconds 10
Start-Sleep -Seconds 10
Start-Sleep -Seconds 10
Write-Host "Fin : $(Get-Date)"
```

**Exécution parallèle** (~10 secondes) :

powershell

```powershell
Write-Host "Début : $(Get-Date)"

$Job1 = Start-Job -ScriptBlock {
    Start-Sleep -Seconds 10
    Write-Host "Job 1 terminé"
}

$Job2 = Start-Job -ScriptBlock {
    Start-Sleep -Seconds 10
    Write-Host "Job 2 terminé"
}

$Job3 = Start-Job -ScriptBlock {
    Start-Sleep -Seconds 10
    Write-Host "Job 3 terminé"
}

# Attendre que tous les jobs soient terminés
Get-Job | Wait-Job

# Récupérer les résultats
Receive-Job $Job1
Receive-Job $Job2
Receive-Job $Job3

# Supprimer les jobs
Remove-Job -Id $Job1.Id, $Job2.Id, $Job3.Id

Write-Host "Fin : $(Get-Date)"
```

### Format standard

**Syntaxe** : `<Verbe>-<Nom>`

Le **verbe** identifie l'action :

- `Get` : Récupère des informations
- `Set` : Modifie une ressource
- `New` : Crée une ressource
- `Remove` : Supprime une ressource

Le **nom** identifie la ressource :

- `Module` : Module PowerShell
- `ADUser` : Utilisateur Active Directory
- `Mailbox` : Boîte aux lettres

### Exemples courants

powershell

```powershell
Get-Location          # Emplacement actuel
Set-Location C:\      # Changer de répertoire
New-Item -ItemType Directory -Name "Test"  # Créer un dossier
Remove-Item "Test"    # Supprimer
Get-ChildItem         # Lister le contenu
Get-Process           # Processus en cours
Get-Service           # Services système
```

### Alias prédéfinis

PowerShell intègre des alias pour faciliter la transition depuis d'autres shells :

|Alias|Cmdlet|Équivalent Unix/Batch|
|---|---|---|
|`ls`|`Get-ChildItem`|Linux `ls`|
|`dir`|`Get-ChildItem`|Batch `dir`|
|`cd`|`Set-Location`|Unix/Batch `cd`|
|`cp`|`Copy-Item`|Unix `cp`|
|`rm`|`Remove-Item`|Unix `rm`|
|`cat`|`Get-Content`|Unix `cat`|

### Get-Help : Commande essentielle

powershell

```powershell
Get-Help Get-Process              # Aide sur une commande
Get-Help Get-Process -Online      # Ouvre la documentation web
Get-Help Get-Process -Examples    # Affiche des exemples
```

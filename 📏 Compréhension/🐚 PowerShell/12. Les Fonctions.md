### Déclaration basique

PowerShell

```powershell
function Hello {
    Write-Host "Bonjour !"
}

Hello  # Appel de la fonction
```

⚠️ **Important** : La fonction doit être déclarée **avant** son appel

### Fonctions avec paramètres

PowerShell

```powershell
function Greet {
    param ([string]$Name)
    Write-Host "Bonjour $Name !"
}

Greet -Name "Alice"
```

### Utilisation de $args

PowerShell

```powershell
function Greet {
    If ($args.Count -gt 0) {
        Write-Host "Bonjour $($args[0])"
    } Else {
        Write-Host "Bonjour à tous !"
    }
}

Greet "Bob"      # Bonjour Bob
Greet            # Bonjour à tous !
```

### Fonctions avancées

**Paramètres obligatoires et validation** :

PowerShell

```powershell
function Conversion {
    param (
        [Parameter(Mandatory=$True)]
        [ValidateRange(0, 255)]
        [int]$Number,
        
        [Parameter(Mandatory=$True)]
        [ValidateSet('Binaire', 'Octal')]
        [string]$Type
    )
    
    Switch ($Type) {
        'Binaire' { [convert]::ToString($Number, 2) }
        'Octal' { [convert]::ToString($Number, 8) }
    }
}

Conversion -Number 10 -Type 'Binaire'  # 1010
```

**Attributs de paramètres courants** :

- `[Parameter(Mandatory=$True)]` : Paramètre obligatoire
- `[ValidateRange(MIN, MAX)]` : Valeur dans un intervalle
- `[ValidateSet('Val1', 'Val2')]` : Liste de valeurs autorisées
- `[Parameter(Mandatory=$False)]` : Paramètre optionnel

### Valeurs de retour

PowerShell

```powershell
function Multiplier {
    param ([int]$a, [int]$b)
    return $a * $b
}

$resultat = Multiplier -a 5 -b 3  # 15
```

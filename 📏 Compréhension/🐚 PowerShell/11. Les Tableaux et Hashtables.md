### Initialisation de tableaux

powershell

```powershell
$Tab = @()                         # Tableau vide
$Tab = @("Lundi", "Mardi")        # Avec valeurs
$Tab = "Lundi", "Mardi", "Mercredi"  # Sans @()
$Tab = 1..5                       # Tableau de 1 à 5
```

### Accès aux éléments

powershell

```powershell
$Tab = @("Lundi", "Mardi", "Mercredi", "Jeudi", "Vendredi")

$Tab[0]           # Premier élément : "Lundi"
$Tab[-1]          # Dernier élément : "Vendredi"
$Tab[1,3]         # Éléments multiples : "Mardi", "Jeudi"
$Tab.Count        # Nombre d'éléments : 5
```

### Ajouter des éléments

powershell

```powershell
$Tab += "Samedi"
$Tab += "Dimanche"
```

### Parcourir un tableau

powershell

```powershell
$Tab = @("Lundi", "Mardi", "Mercredi")
$Count = 1

Foreach ($Element in $Tab) {
    Write-Host "Jour N°$Count : $Element"
    $Count++
}
```

### Hashtables (Tableaux associatifs)

**Clé/Valeur** :

powershell

```powershell
$Hash = @{
    1 = "Lundi"
    2 = "Mardi"
    3 = "Mercredi"
}

$Hash[2]                  # "Mardi"
$Hash.Keys                # Liste des clés
$Hash.Values              # Liste des valeurs
$Hash.Count               # Nombre d'éléments
$Hash.Add(4, "Jeudi")     # Ajouter
$Hash.Remove(1)           # Supprimer
$Hash.GetEnumerator()     # Parcourir
```

**Parcourir une hashtable** :

powershell

```powershell
$Hash.GetEnumerator() | ForEach-Object {
    Write-Host "Clé: $($_.Key), Valeur: $($_.Value)"
}
```

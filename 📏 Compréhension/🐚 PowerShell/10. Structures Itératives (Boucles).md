### Opérateurs unaires

powershell

```powershell
$i = 1
$i++      # Post-incrémentation : affiche 1, puis incrémente à 2
++$i      # Pré-incrémentation : incrémente à 2, puis affiche 2
$i--      # Post-décrémentation
--$i      # Pré-décrémentation
```

**Exemple** :

powershell

```powershell
$i = 1
Write-Host "i vaut $(($i++))"    # Affiche 1, i devient 2
Write-Host "i vaut maintenant $i" # Affiche 2

$j = 1
Write-Host "j vaut $((++$j))"    # Incrémente d'abord, affiche 2
```

### Boucle For

powershell

```powershell
For ($i = 0; $i -le 10; $i++) {
    Write-Host "Valeur : $i"
}
```

**Conditions multiples** :

powershell

```powershell
For (($i=0), ($j=0); $i -le 20 -and ($i+$j) -le 15; $i++, $j++) {
    Write-Host "i: $i, j: $j, somme: $($i+$j)"
}
```

### Boucle Foreach

**Syntaxe classique** :

powershell

```powershell
$Services = Get-Service
Foreach ($Service in $Services) {
    Write-Host "$($Service.Name) --> $($Service.Status)"
}
```

**Dans un pipeline** :

powershell

```powershell
Get-Service | ForEach-Object {
    Write-Host "$($_.Name) --> $($_.Status)"
}

# Alias : % au lieu de ForEach-Object
Get-Service | % { Write-Host "$($_.Name)" }
```

### Boucle While

powershell

```powershell
$Count = 0
While ($Count -le 10) {
    Write-Host "Compteur : $Count"
    $Count++
}
```

### Boucle Do While

Condition testée **à la fin** (au moins 1 passage garanti) :

powershell

```powershell
$Count = 0
Do {
    Write-Host "Compteur : $Count"
    $Count++
} While ($Count -le 10)
```

### Boucle Do Until

Exécute **jusqu'à ce que** la condition soit vraie :

powershell

```powershell
$Count = 0
Do {
    Write-Host "Compteur : $Count"
    $Count++
} Until ($Count -eq 10)
```

### Break et Continue

powershell

```powershell
# Break : Sort complètement de la boucle
For ($i = 0; $i -le 10; $i++) {
    If ($i -eq 5) { break }
    Write-Host $i
}  # Affiche 0, 1, 2, 3, 4

# Continue : Passe à l'itération suivante
For ($i = 0; $i -le 10; $i++) {
    If ($i -eq 5) { continue }
    Write-Host $i
}  # Affiche tout sauf 5
```

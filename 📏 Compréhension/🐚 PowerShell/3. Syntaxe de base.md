### Sensibilité

PowerShell **n'est PAS sensible** :

- À la **casse** (majuscules/minuscules)
- Aux **espaces** et **tabulations**

powershell

```powershell
wRite-ouTput "Ceci fonctionne parfaitement"
```

### Caractères d'échappement

Symbole : `` ` `` (backtick/accent grave)

|Séquence|Effet|
|---|---|
|`` `n ``|Nouvelle ligne|
|`` `t ``|Tabulation|
|`` `$ ``|Affiche `$` littéralement|

powershell

```powershell
Write-Output "`nSaut de ligne`nTabulation `tici"
```

⚠️ **Attention** : Ne pas confondre `` ` `` (backtick) avec `'` (apostrophe)

### Quotes et Double Quotes

**Single quotes** (`'`) : Texte littéral

powershell

```powershell
$i = 5
'The value of $i is $i'  # Affiche : The value of $i is $i
```

**Double quotes** (`"`) : Interprète les variables

powershell

```powershell
$i = 5
"The value of $i is $i"  # Affiche : The value of 5 is 5
```

### Interpolation de variables

powershell

```powershell
$Host = Get-Host
Write-Host "La version est $($Host.Version)"
```

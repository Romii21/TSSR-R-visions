### Opérateurs de comparaison

|Opérateur|Signification|
|---|---|
|`-eq`|Égal à (equal to)|
|`-ne`|Différent de (not equal)|
|`-gt`|Plus grand que (greater than)|
|`-lt`|Plus petit que (less than)|
|`-ge`|Plus grand ou égal (greater or equal)|
|`-le`|Plus petit ou égal (less or equal)|

### Opérateurs de correspondance

|Opérateur|Description|
|---|---|
|`-like` / `-notLike`|Avec wildcards (`*`, `?`)|
|`-match` / `-notMatch`|Avec regex|

powershell

```powershell
"identique" -eq "identique"     # True
"identique" -ne "différent"     # True
5 -gt 3                         # True
"hello" -like "hel*"            # True
"123" -match "\d"               # True
```

### Opérateurs logiques

|Opérateur|Description|
|---|---|
|`-and`|ET logique|
|`-or`|OU logique|
|`-xor`|OU exclusif|
|`-not` ou `!`|NON logique|

powershell

```powershell
$trois = 3
($trois -eq 3) -and ($trois -lt 5)    # True
($trois -eq 3) -or ($trois -eq 4)     # True
($trois -eq 3) -xor ($trois -lt 4)    # False (les deux sont vrais)
!($trois -eq 3)                       # False
```

### Tests sur les chaînes

powershell

```powershell
[String]::IsNullOrEmpty($variable)    # True si vide
![String]::IsNullOrEmpty($variable)   # True si non vide
```

### Tests sur les chemins

powershell

```powershell
Test-Path -Path "C:\Windows"          # True si existe
Test-Path -Path "C:\Inexistant"       # False
```

### Structure If/Else

powershell

```powershell
If (condition) {
    # Instructions si vrai
} Else {
    # Instructions si faux
}
```

**Exemple** :

powershell

```powershell
If (Test-Path "C:\Temp") {
    Write-Host "Le dossier existe"
} Else {
    Write-Host "Le dossier n'existe pas"
    New-Item -Path "C:\Temp" -ItemType Directory
}
```

### Structure If/ElseIf/Else

powershell

```powershell
$nombre = 10

If ($nombre -lt 0) {
    Write-Host "Négatif"
} ElseIf ($nombre -eq 0) {
    Write-Host "Zéro"
} Else {
    Write-Host "Positif"
}
```

### Structure Switch

powershell

```powershell
$valeur = 5

Switch ($valeur) {
    1 { Write-Host "Un" }
    2 { Write-Host "Deux" }
    5 { Write-Host "Cinq" }
    default { Write-Host "Autre valeur" }
}
```

**Avec break** :

powershell

```powershell
Switch ($Service.Status) {
    "Stopped" {
        Write-Host "Service arrêté" -ForegroundColor Red
        break
    }
    "Running" {
        Write-Host "Service en cours" -ForegroundColor Green
        break
    }
    default {
        Write-Host "Statut inconnu" -ForegroundColor Blue
    }
}
```

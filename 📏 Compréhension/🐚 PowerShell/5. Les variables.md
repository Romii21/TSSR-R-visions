### Déclaration et utilisation

**Syntaxe** : `$NomVariable = valeur`

powershell

```powershell
$Var = "Bonjour"
Write-Host $Var           # Affiche : Bonjour

$Var1 = $Var
$Var = "Hello"
Write-Host "$Var1 et $Var"  # Affiche : Bonjour et Hello
```

### Règles de nommage

✅ Commence par `$`  
✅ Lettres, chiffres, caractères spéciaux autorisés  
✅ Non sensible à la casse  
✅ Convention : **PascalCase** (`$MonNomDeVariable`)  
❌ Ne doit pas être un mot-clé du langage

### Types de données

powershell

```powershell
[int]       # Entier 32 bits
[string]    # Chaîne de caractères
[bool]      # Booléen
[double]    # Nombre décimal
[array]     # Tableau
```

**Connaître le type** :

powershell

```powershell
"Texte".GetType()     # Retourne : String
(10).GetType()        # Retourne : Int32
```

### Transtypage (Casting)

powershell

```powershell
[int]$Nombre = "10"
$Nombre + 5           # Résultat : 15 (addition numérique)
```

### Méthodes sur les variables

powershell

```powershell
"hello".ToUpper()               # "HELLO"
"  hello  ".Trim()              # "hello" (supprime espaces)
"Hello boy".Replace("boy", "world")  # "Hello world"
```

### Substitution de commandes

Stocker le résultat d'une commande :

powershell

```powershell
$HostVersion = $(Get-Host | Select-Object Version)
$AllProcess = Get-Process
```

### Substitution arithmétique

powershell

```powershell
$Total = 10 + 2        # 12
Write-Host $(12 * 6)   # 72
```

### Portée des variables

|Portée|Description|
|---|---|
|**Global**|Disponible partout (console, tous scripts)|
|**Script**|Limité au script en cours|
|**Local**|Limité au bloc actuel|
|**Private**|Non visible hors de sa portée|

**Modificateurs** :

powershell

```powershell
$global:Variable = "Valeur globale"
$script:Variable = "Valeur script"
$local:Variable = "Valeur locale"
$private:Variable = "Valeur privée"
```

### Variables spéciales

|Variable|Description|
|---|---|
|`$?`|État de la dernière commande (True/False)|
|`$_`|Objet actuel dans un pipeline|
|`$args`|Paramètres passés à une fonction/script|
|`$null`|Valeur vide/absente|
|`$true` / `$false`|Booléens|
|`$Home`|Répertoire personnel de l'utilisateur|
|`$Profile`|Chemin du profil PowerShell|
|`$PSCulture`|Langue du système|

### Variables d'environnement

Préfixe : `$env:`

powershell

```powershell
$env:PROCESSOR_ARCHITECTURE    # Architecture CPU
$env:USERNAME                  # Nom d'utilisateur
$env:COMPUTERNAME              # Nom de l'ordinateur
$env:PATH                      # Variable PATH système

Get-ChildItem env:             # Liste toutes les variables d'environnement
```

### Suppression de variables

powershell

```powershell
Remove-Variable Var
Remove-Variable Var1, Var2
```

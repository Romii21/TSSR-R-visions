### Prérequis

- Service **WinRM** démarré sur la machine distante
- Droits d'administration sur l'ordinateur cible

### Cmdlets avec ComputerName

Certaines cmdlets acceptent le paramètre `-ComputerName` :

powershell

```powershell
Test-Connection -ComputerName "client1"
Restart-Computer -ComputerName "serveur2"
Stop-Computer -ComputerName "client1"
Get-Service -ComputerName "serveur3"
```

### Session interactive

powershell

```powershell
Enter-PSSession -ComputerName "client1"
# Vous êtes maintenant sur client1
[client1]: PS C:\> Get-ChildItem C:\

Exit-PSSession  # Fermer la session
```

### Exécution de commandes

powershell

```powershell
Invoke-Command -ComputerName "client1" -ScriptBlock {
    Get-ChildItem C:\
    New-Item -Path "C:\test" -ItemType Directory
}
```

**Exemple avec variable** :

powershell

```powershell
$computers = "client1", "client2", "serveur1"
Invoke-Command -ComputerName $computers -ScriptBlock {
    Get-Service | Where-Object {$_.Status -eq "Running"}
}
```

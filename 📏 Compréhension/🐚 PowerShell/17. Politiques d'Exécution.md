### Stratégies d'exécution

|Stratégie|Description|
|---|---|
|**Restricted**|Aucun script autorisé (défaut client)|
|**AllSigned**|Uniquement scripts signés|
|**RemoteSigned**|Scripts locaux OK, scripts Internet signés (défaut serveur)|
|**Unrestricted**|Tous scripts, confirmation pour scripts Internet|
|**Bypass**|Tous scripts sans avertissement|
|**Undefined**|Aucune stratégie définie|
|**Default**|Équivalent à Restricted ou RemoteSigned selon le système|

### Étendues

|Étendue|Description|
|---|---|
|**MachinePolicy**|Stratégie de groupe pour tous les utilisateurs|
|**UserPolicy**|Stratégie de groupe pour l'utilisateur actuel|
|**Process**|Session PowerShell actuelle uniquement|
|**CurrentUser**|Utilisateur actuel|
|**LocalMachine**|Tous les utilisateurs (défaut)|

### Commandes

**Consulter la stratégie** :

powershell

```powershell
Get-ExecutionPolicy
Get-ExecutionPolicy -List  # Toutes les étendues
```

**Modifier la stratégie** :

powershell

```powershell
# LocalMachine (tous les utilisateurs)
Set-ExecutionPolicy -Scope LocalMachine -ExecutionPolicy Unrestricted

# CurrentUser (utilisateur actuel)
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy Bypass

# Remettre à Undefined
Set-ExecutionPolicy -Scope LocalMachine -ExecutionPolicy Undefined
```

⚠️ **Attention** : Nécessite des droits d'administration pour modifier LocalMachine

### Bypass temporaire

powershell

```powershell
# Exécuter un script en bypassant la politique
PowerShell.exe -ExecutionPolicy Bypass -File "C:\script.ps1"
```

### Commandes de navigation

powershell

```powershell
Get-Location                   # Affiche l'emplacement actuel
Set-Location -Path C:\         # Change le répertoire
Get-ChildItem                  # Liste le contenu
Get-ChildItem -Recurse         # Liste récursivement
Get-ChildItem -Path *Folder*   # Filtre par nom
```

### Créer des dossiers et fichiers

powershell

```powershell
# Créer un dossier
New-Item -Path "Temp" -ItemType Directory

# Créer un fichier
New-Item -Path "file1.txt" -ItemType File

# Créer plusieurs fichiers
1..5 | ForEach-Object { New-Item -Path "file$_.txt" -ItemType File }
```

### Déplacer et copier

powershell

```powershell
# Déplacer
Move-Item -Path "file1.txt" -Destination "Dossier1"

# Copier
Copy-Item -Path "file1.txt" -Destination "file1_copie.txt"

# Copier récursivement
Copy-Item -Path "Dossier1" -Destination "Dossier2" -Recurse
```

### Supprimer

powershell

```powershell
# Supprimer un fichier
Remove-Item -Path "file1.txt"

# Supprimer un dossier non vide
Remove-Item -Path "Dossier1" -Recurse

# Supprimer avec confirmation
Remove-Item -Path "file2.txt" -Confirm
```

⚠️ **ATTENTION** : Pas de corbeille, suppression définitive !

### Lire le contenu d'un fichier

powershell

```powershell
Get-Content -Path "fichier.txt"
Get-Content -Path "fichier.txt" | Select-Object -First 10  # 10 premières lignes
```

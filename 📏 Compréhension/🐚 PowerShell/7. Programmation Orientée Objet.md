### Concepts fondamentaux

En PowerShell, **tout est objet** !

**Classe** : Modèle/plan pour créer des objets  
**Attributs** : Caractéristiques (propriétés)  
**Méthodes** : Actions possibles  
**Objet** : Instance d'une classe

### Analogie avec une voiture

|Concept|Exemple|
|---|---|
|**Classe**|Voiture|
|**Attributs**|Couleur, Vitesse, Nombre de roues|
|**Méthodes**|Rouler(), Freiner(), Klaxonner()|

### Manipulation d'objets

powershell

```powershell
# Récupérer un objet
$file = Get-ChildItem "fichier.txt"

# Accéder aux propriétés
$file.Name
$file.Length
$file.CreationTime

# Utiliser une méthode
$file.CopyTo("nouveau_fichier.txt")
$file.Delete()
```

### Get-Member : Explorer un objet

powershell

```powershell
Get-ChildItem | Get-Member                 # Voir propriétés et méthodes
Get-ChildItem | Get-Member -MemberType Property  # Seulement les propriétés
```

### GetType() : Connaître le type

powershell

```powershell
(Get-Host).GetType()
# IsPublic IsSerial Name         BaseType
# -------- -------- ----         --------
# False    False    InternalHost System.Management.Automation.Host.PSHost
```

### Pipeline et objets

powershell

```powershell
Get-Host | Select-Object -Property Version
# Version
# -------
# 5.1.22000.613
```

**Explication** :

1. `Get-Host` retourne un objet
2. Le pipe `|` envoie cet objet à la commande suivante
3. `Select-Object -Property Version` sélectionne uniquement la propriété `Version`

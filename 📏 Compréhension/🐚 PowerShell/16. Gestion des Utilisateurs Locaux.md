### Méthode ADSI

**ADSI** (Active Directory Service Interfaces) permet d'accéder à la base locale des comptes :

powershell

```powershell
$Local = [ADSI]"WinNT://."  # Base locale du système
```

**Nomenclature** :

- Le point `.` représente la machine locale
- Peut être remplacé par le nom de l'ordinateur
- Nécessite des droits d'administration pour les opérations distantes

### Vérifier l'existence d'un compte

**Méthode native** :

powershell

```powershell
Get-LocalUser -Name "wilder"
```

**Méthode ADSI** :

powershell

```powershell
$Nom = "wilder"
$Compte = [ADSI]"WinNT://./$Nom"

If ($Compte.Path) {
    Write-Host "Le compte $($Compte.FullName) existe" -ForegroundColor Green
} Else {
    Write-Host "Le compte $Nom n'existe pas" -ForegroundColor Red
}
```

### Créer un compte local

**Méthode ADSI** :

powershell

````powershell
$Local = [ADSI]"WinNT://."
$Nom = Read-Host -Prompt "Nom de compte"
$Description = Read-Host -Prompt "Description"

$Compte = [ADSI]"WinNT://./$Nom"
If (!$Compte.Path) {
    $Utilisateur = $Local.Create("User", $Nom)
    $Utilisateur.InvokeSet("Description", $Description)
    $Utilisateur.CommitChanges()
    Write-Host "$Nom créé avec succès" -ForegroundColor Green
} Else {
    Write-Host "$Nom existe déjà" -ForegroundColor Red
}
```

### Créer plusieurs comptes depuis un fichier

**Format du fichier** (`comptes.txt`) :
```
"jhenry"/"Jean Henry"/"Comptable"
"dlarry"/"David Larry"/"Comptable"
"jlanvin"/"Jerome Lanvin"/"Secrétaire"
````

**Script de création** :

powershell

```powershell
$Local = [ADSI]"WinNT://."
$Fichier = "C:\comptes.txt"

If (Test-Path $Fichier) {
    $Lignes = Get-Content -Path $Fichier
    Foreach ($Ligne in $Lignes) {
        $TabCompte = $Ligne.Split("/")
        $Nom = $TabCompte[0].Trim('"')
        $NomComplet = $TabCompte[1].Trim('"')
        $Description = $TabCompte[2].Trim('"')
        
        $Compte = [ADSI]"WinNT://./$Nom"
        If (!$Compte.Path) {
            $Utilisateur = $Local.Create("User", $Nom)
            $Utilisateur.InvokeSet("FullName", $NomComplet)
            $Utilisateur.InvokeSet("Description", $Description)
            $Utilisateur.CommitChanges()
            Write-Host "$Nom créé" -ForegroundColor Green
        } Else {
            Write-Host "$Nom existe déjà" -ForegroundColor Yellow
        }
    }
}
```

### Supprimer des comptes

powershell

```powershell
$Local = [ADSI]"WinNT://."
$Fichier = "C:\comptes.txt"

If (Test-Path -Path $Fichier) {
    $Lignes = Get-Content -Path $Fichier
    Foreach ($Ligne in $Lignes) {
        $TabCompte = $Ligne.Split("/")
        $Nom = $TabCompte[0].Trim('"')
        
        $Compte = [ADSI]"WinNT://./$Nom"
        If ($Compte.Path) {
            $Local.Delete("user", $Nom)
            Write-Host "$Nom supprimé" -ForegroundColor Green
        } Else {
            Write-Host "$Nom n'existe pas" -ForegroundColor Red
        }
    }
}
```

## 🎯 Points Clés à Retenir

✅ **PowerShell manipule des objets .NET**, pas du simple texte  
✅ **Les cmdlets suivent le format Verbe-Nom** (`Get-Service`, `Set-Location`)  
✅ **Les variables commencent toujours par `$`** et ont une portée définie  
✅ **Les accolades `{ }` délimitent les blocs** de code (if, for, fonctions)  
✅ **Le pipeline `|` permet de chaîner les commandes** et passer des objets  
✅ **PowerShell intègre des alias** pour les commandes Unix/Batch (`ls`, `cd`, `cp`)  
✅ **Les objets ont des propriétés et des méthodes** accessibles avec la notation `.`  
✅ **L'exécution à distance nécessite WinRM** et des droits d'administration  
✅ **Les jobs permettent l'exécution parallèle** de tâches longues  
✅ **Les politiques d'exécution contrôlent la sécurité** des scripts

## 🔧 Exemples Pratiques de Scripts

### Script de sauvegarde

powershell

```powershell
# Sauvegarde d'un dossier
$Source = "C:\Documents"
$Destination = "D:\Backup\Documents_$(Get-Date -Format 'yyyyMMdd')"

If (!(Test-Path $Destination)) {
    New-Item -Path $Destination -ItemType Directory
}

Copy-Item -Path "$Source\*" -Destination $Destination -Recurse -Force
Write-Host "Sauvegarde terminée : $Destination" -ForegroundColor Green
```

### Script de nettoyage

powershell

```powershell
# Supprimer les fichiers temporaires de plus de 7 jours
$Path = "C:\Temp"
$Days = 7

Get-ChildItem -Path $Path -Recurse |
    Where-Object {$_.LastWriteTime -lt (Get-Date).AddDays(-$Days)} |
    Remove-Item -Force -Recurse

Write-Host "Nettoyage terminé" -ForegroundColor Green
```

### Script d'inventaire système

powershell

```powershell
# Informations système
$Info = @{
    Ordinateur = $env:COMPUTERNAME
    Utilisateur = $env:USERNAME
    OS = (Get-CimInstance Win32_OperatingSystem).Caption
    Version = (Get-CimInstance Win32_OperatingSystem).Version
    Memoire = "$([Math]::Round((Get-CimInstance Win32_ComputerSystem).TotalPhysicalMemory / 1GB, 2)) GB"
    Processeur = (Get-CimInstance Win32_Processor).Name
}

$Info | Format-Table -AutoSize
$Info | Export-Csv -Path "C:\inventaire.csv" -NoTypeInformation
```

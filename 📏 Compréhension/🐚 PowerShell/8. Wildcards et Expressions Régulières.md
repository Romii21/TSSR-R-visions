### Wildcards (Caractères génériques)

|Symbole|Signification|
|---|---|
|`*`|Zéro ou plusieurs caractères|
|`?`|Exactement un caractère|

powershell

```powershell
Get-ChildItem C:\*.txt           # Tous les fichiers .txt
Get-ChildItem C:\file?.txt       # file1.txt, file2.txt, etc.
Get-ChildItem | Where-Object {$_.Name -like "*program*"}
```

### Expressions Régulières (Regex)

Utilisées avec `-match`, `-replace`, `Select-String`

powershell

```powershell
"123" -match "\d"                # True (\d = chiffre)
"Hello" -match "^H"              # True (^ = début de chaîne)
"Hello" -match "o$"              # True ($ = fin de chaîne)

"Hello boy" -replace "boy", "everybody"  # "Hello everybody"
"Hello boy" -replace "(boy|girl)", "everybody"  # Capture de groupes
```

**Métacaractères courants** :

- `\d` : Chiffre
- `\w` : Caractère alphanumérique
- `^` : Début de chaîne
- `$` : Fin de chaîne
- `.` : N'importe quel caractère
- `*` : 0 ou plusieurs occurrences
- `+` : 1 ou plusieurs occurrences
